Sobre o Projeto

Uma aplicação interativa para explorar dados globais.

O Desafio

Este projeto foi desenvolvido como requisito avaliativo da disciplina de Programação Web na Universidade Estadual do Piauí (UESPI).

O objetivo principal foi criar uma aplicação Single Page Application (SPA) simulada, consumindo a REST Countries API para exibir dados em tempo real, implementar filtros dinâmicos e gerenciar favoritos via LocalStorage.

Stack Tecnológica

HTML5 Semântico CSS3 Moderno JavaScript ES6+ Leaflet.js Fetch API
